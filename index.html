스티커 메모
사용법 안내

1메모

  • ＋ 검정 버튼 — 새 메모 추가
  • 테이프(상단 가운데) 드래그 — 메모 이동
  • 테이프 Ctrl+드래그 — 제목 복사된 새 메모 생성
  • 본문 클릭 — 본문 전체 복사 + 리플 애니메이션
  • 제목/본문 더블클릭 — 수정 모드
  • 본문 텍스트 드래그 — 일부만 선택 (Ctrl+C로 복사)
  • 우측 하단 ↘ — 크기 조절 (잠긴 상태에서도 가능)

2수정 · 잠금 · 삭제

  • 🔓 자물쇠 — 위치 잠금 (크기·편집은 가능)
  • ✏ 연필 — 제목 + 본문 편집 모드
  • × 닫기 — 메모 삭제
  • 편집 중 Tab — 제목 → 본문으로 이동
  • 편집 중 Esc 또는 Ctrl+Enter — 편집 종료 (저장)

3여러 개 선택 · 이동

  • 빈 공간 드래그 — Rubber-band 선택
  • Shift + 클릭 — 개별 선택 + 맨 위로 올리기
  • 선택된 항목 드래그 — 전부 같이 이동
  • Delete — 일괄 삭제 · Esc — 해제

4탭

  • ＋ 탭 추가 — 새 주제
  • 탭 우클릭 / 길게 터치 — 이름·색·삭제 메뉴
  • 전체 탭 — 모든 메모 제목 목록 (헤더 클릭해 펼치기)

5테이프 라벨 (구역 표시)

  • ＋ 베이지 버튼 — 새 라벨 추가
  • 라벨 어디든 드래그로 이동, 클릭해서 이름 편집
  • 현재 탭에만 보임, 8색 형광펜 팔레트

6저장 · 동기화

  • 모든 변경 자동 저장 (localStorage)
  • 💾 또는 Ctrl+S — 수동 저장
  • Ctrl+Z — 실행 취소 · Ctrl+Y — 다시 실행
  • ⚙ 설정에서 GitHub 토큰 → Gist로 기기 간 동기화
※ 모바일에서는 일부 기능 (Rubber-band, Shift)이 제한됩니다

설정

GitHub Gist에 메모를 저장하면 다른 기기에서도 열어볼 수 있어요.
토큰 발급하기 →  필요한 권한: gist
비어있으면 "새로 생성" 버튼으로 프라이빗 Gist를 만들어줍니다. 다른 기기에서는 같은 Gist ID를 입력하세요.
