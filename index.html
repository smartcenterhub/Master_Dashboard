바로가기 편집

제목은 자유, URL은 클릭 시 복사됩니다.
사용법 안내

1메모

  • ＋ 검정 버튼 또는 Ctrl+N — 새 메모 추가
  • 테이프(상단 가운데) 드래그 — 메모 이동
  • 테이프 Ctrl+드래그 — 제목 복사된 새 메모 생성
  • 본문 클릭 — 본문 전체 복사 + 리플 애니메이션
  • 제목/본문 더블클릭 — 수정 모드
  • 본문 텍스트 드래그 — 일부만 선택 (Ctrl+C로 복사)
  • 우측 하단 ↘ — 크기 조절 (잠긴 상태에서도 가능)

2수정 · 잠금 · 삭제

  • 🔓 자물쇠 — 위치 잠금 (크기·편집은 가능)
  • ✏ 연필 — 제목 + 본문 편집 모드
  • × 닫기 — 메모 삭제
  • 편집 중 Tab — 제목 → 본문으로 이동
  • 편집 중 Esc 또는 Ctrl+Enter — 편집 종료 (저장)

3여러 개 선택 · 이동

  • 빈 공간 드래그 — Rubber-band 선택
  • Shift + 클릭 — 개별 선택 + 맨 위로 올리기
  • 선택된 항목 드래그 — 전부 같이 이동
  • Ctrl+C / Ctrl+V — 선택 항목 복제 (메모·라벨·황동)
  • Delete — 일괄 삭제 · Esc — 해제

4탭

  • ＋ 탭 추가 — 새 주제
  • 탭 우클릭 / 길게 터치 — 이름·색·삭제 메뉴
  • 탭 드래그 — 순서 변경
  • 전체 탭 — 모든 메모 제목 목록 (헤더 클릭해 펼치기)
  • 하위탭 (사이드바) — 왼쪽 사이드바에서 세부 분류 · 항목 클릭해 전환
  • 사이드바 ＋ 하위탭 추가 · 하위탭 우클릭으로 이름/삭제
  • 하위탭 드래그로 순서 변경 · 노란 포스트잇 핸들로 사이드바 접기/펼치기

5테이프 라벨 (구역 표시)

  • ＋ 베이지 버튼 — 색 선택 팝업 → 색 고르면 라벨 생성
  • 라벨 어디든 드래그로 이동
  • 더블클릭으로 이름 편집, 외부 클릭 또는 Esc로 저장
  • 🔒 자물쇠로 위치 고정 · Shift+클릭 후 Delete로 삭제
  • 현재 탭에만 보임, 8색 형광펜 팔레트 (생성 후 색 변경 불가)

6바로가기 (황동 명패)

  • ＋ 황동 버튼 — 제목 + URL 입력해서 생성
  • 단일 클릭 — URL 클립보드에 복사 (+ 초록 플래시)
  • 더블클릭 — 제목·URL·부가 텍스트 편집 (삭제 버튼도 있음)
  • 드래그로 이동, Shift+클릭 → Delete로도 삭제

7저장 · 동기화

  • 모든 변경 자동 저장 (localStorage)
  • 💾 또는 Ctrl+S — 수동 저장
  • Ctrl+Z — 실행 취소 · Ctrl+Y — 다시 실행
  • ⚙ 설정에서 GitHub 토큰 → Gist로 기기 간 동기화
※ 모바일에서는 일부 기능 (Rubber-band, Shift)이 제한됩니다

설정

GitHub Gist에 메모를 저장하면 다른 기기에서도 열어볼 수 있어요.
토큰 발급하기 →  필요한 권한: gist
비어있으면 "새로 생성" 버튼으로 프라이빗 Gist를 만들어줍니다. 다른 기기에서는 같은 Gist ID를 입력하세요.
